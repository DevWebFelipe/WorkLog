# WorkLog

## 📌 Sobre o projeto

O **WorkLog** é um projeto pessoal desenvolvido com o objetivo de gerenciar atividades do dia a dia, como:

* Anotações
* Apontamentos
* Tickets

Este projeto nasceu inicialmente como uma ferramenta de uso próprio, mas está sendo **reconstruído e evoluído com foco em estudo e melhoria contínua**.

---

## 🎯 Objetivo

Além da utilidade prática, o projeto tem como principais objetivos:

* Praticar boas práticas de desenvolvimento
* Evoluir arquitetura de aplicações Delphi
* Estruturar um projeto escalável e de fácil manutenção
* Experimentar novas abordagens e tecnologias

---

## 🧪 Tecnologias e conceitos em estudo

O projeto também serve como ambiente de testes para:

* ⚙️ **Delphi 12.1**
* 🚀 **Horse (API REST)**
* 🗄️ Integração com banco de dados
* 🧱 Padrões de arquitetura (DAO, VO, organização em camadas)
* 🔄 Evolução para modelo cliente-servidor

---

## 🏗️ Estrutura e evolução

O WorkLog está em constante evolução. Algumas melhorias planejadas:

* Sistema de autenticação (login por usuário)
* Separação de dados por usuário
* Criação de API para desacoplamento do front-end
* Possível migração futura para aplicação web
* Melhorias na arquitetura e reutilização de código (Libs, BaseDAO, BaseVO)

---

## ⚠️ Observações

* Este é um projeto **em desenvolvimento ativo**
* Estruturas, padrões e organização podem mudar com o tempo
* Parte das implementações pode existir com foco em aprendizado, não necessariamente na forma final ideal
